Gemini Clone


A simple AI chatbot clone built using React and Vite with the Groq API.


Features

Chat interface

AI response generation

Responsive UI

Fast performance using Vite




Tech Stack

React

Vite

JavaScript

Groq API

Installation
git clone https://github.com/KARTIKP124/gemini-clone.git

cd gemini-clone

npm install

npm run dev

Environment Variables

Create a .env file in the root folder and add:

VITE_GROQ_API_KEY=your_api_key

Build

npm run build
